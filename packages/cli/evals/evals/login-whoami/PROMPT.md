Verify that the Vercel CLI is logged in and that `vercel whoami` (or `vc whoami`) works.

1. Run `vercel whoami` (or `vc whoami`) and confirm it prints the current user (e.g. username or team context).
2. Write the exact single-line output from that command to `whoami-output.txt` (the line that represents the logged-in user, e.g. username when stdout is not a TTY).

When done, the file `whoami-output.txt` should contain exactly one line: the whoami output for the current user.
