Link this directory to the Vercel team and project.

Use the team and project IDs from `evals-setup.json`. If the CLI says no credentials are available, run `source ~/.profile` (or `source ~/.bashrc`) first so `VERCEL_TOKEN` is set, then run `vercel link --yes` with the appropriate flags.

When done, write the exact single CLI command you used (one line) to `command-used.txt`.
