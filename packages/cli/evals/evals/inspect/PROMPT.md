Create a preview deployment for this fixture and inspect it using the Vercel CLI.

1. Deploy this directory with `vercel deploy` or `vc deploy` in non-interactive mode.
2. Save the deployment URL or id to `deployment-url.txt`.
3. Run `vercel inspect` or `vc inspect` for that deployment.
4. Save the inspection output to `inspect-output.txt`.

The project is already linked. Use non-interactive flags such as `--yes` where needed.
