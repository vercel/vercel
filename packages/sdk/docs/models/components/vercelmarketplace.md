# VercelMarketplace

Will be used to create an invoice item. The price must be in cents: 2000 for $20.

## Example Usage

```typescript
import { VercelMarketplace } from "@vercel/sdk/models/components";

let value: VercelMarketplace = {
  price: 9980.26,
  quantity: 2436.78,
  hidden: false,
};
```
