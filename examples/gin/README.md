# Gin Starter

Deploy your [Gin](https://gin-gonic.com/) project to Vercel with zero configuration.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?demo-description=Deploy%20Go%20Gin%20applications%20with%20zero%20configuration.&demo-title=Gin%20Boilerplate&demo-url=https%3A%2F%2Fvercel-plus-gin.vercel.app%2F&from=templates&project-name=Gin%20Boilerplate&repository-name=gin-go-boilerplate&repository-url=https%3A%2F%2Fgithub.com%2Fvercel%2Fvercel%2Ftree%2Fmain%2Fexamples%2Fgin&skippable-integrations=1)

_Live Example: https://vercel-plus-gin.vercel.app/_

Visit the [Gin documentation](https://gin-gonic.com/docs/) to learn more.

## Getting Started

Make sure you have Go installed. If not, install it from [go.dev](https://go.dev/dl/).

Build the project:

```bash
go build
```

## Running Locally

Start the development server on http://localhost:3000

```bash
go run main.go
```

When you make changes to your project, restart the server to see your changes.

## Deploying to Vercel

Deploy your project to Vercel with the following command:

```bash
npm install -g vercel
vercel --prod
```

Or `git push` to your repository with our [git integration](https://vercel.com/docs/deployments/git).

To view the source code for this template, [visit the example repository](https://github.com/vercel/vercel/tree/main/examples/gin).
