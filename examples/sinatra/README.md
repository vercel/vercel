Deploy your [Sinatra](https://sinatrarb.com/) project to Vercel with zero configuration.

_Live Example: https://vercel-plus-sinatra.vercel.app/_

Visit the [Sinatra documentation](https://sinatrarb.com/) to learn more.

## Getting Started

Install the required dependencies:

```bash
bundle install
```

## Running Locally

Start the development server on http://localhost:9292

```bash
bundle exec rackup
```

When you make changes to your project, restart the server to see your changes.

## Deploying to Vercel

Deploy your project to Vercel with the following command:

```bash
npm install -g vercel
vercel --prod
```

Or `git push` to your repository with our [git integration](https://vercel.com/docs/deployments/git).

To view the source code for this template, [visit the example repository](https://github.com/vercel/vercel/tree/main/examples/sinatra).
