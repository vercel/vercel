# vercel open

Open the current project in the Vercel Dashboard.

## Synopsis

```bash
vercel open
```

## Description

The `open` command opens your project's Vercel Dashboard page in your default web browser.

## Examples

### Open Project Dashboard

```bash
vercel open
```

Opens: `https://vercel.com/<team>/<project>`

---

## Requirements

- Project must be linked (`vercel link`)
- Browser must be available
