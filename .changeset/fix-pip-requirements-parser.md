Add regression tests for PEP 508 environment markers with chained `and`/`or` operators.

The `pip-requirements-js` dependency was updated to version 1.0.3, which fixes parsing of complex
environment markers like those generated by Poetry. This changeset adds regression tests to ensure
proper parsing of requirements like:

```
greenlet==3.3.0 ; python_version == "3.12" and (platform_machine == "aarch64" or platform_machine == "ppc64le" or platform_machine == "x86_64")
```
